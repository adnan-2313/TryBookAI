# TryBookAI

TryBookAI is a web application that leverages advanced AI technology to generate captivating books. It features a user-friendly interface, dynamic theme switching, and sections that showcase the app's functionality, future roadmap, and user testimonials. 

## Features

- **AI-Powered Book Generation**: Create engaging books with the help of AI.
- **Interactive Sections**:
  - **How It Works**: A detailed explanation of how the platform operates.
  - **Roadmap**: A look at upcoming features and developments for BookAI.
- **User Testimonials**: Read what our users have to say about their experience.
- **Newsletter Subscription**: Sign up to receive the latest AI writing tips and updates.
- **Dynamic Theme Switching**: Easily toggle between light and dark themes.

## Getting Started

### Prerequisites

Before you begin, ensure you have met the following requirements:

- Node.js and npm installed on your development machine.

### Installation

1. Clone the repository:

    ```bash
    git clone git@github.com:adnan-2313/TryBookAI.git
    ```

2. Navigate to the project directory:

    ```bash
    cd TryBookAI
    ```

3. Install the dependencies:

    ```bash
    npm install
    ```

### Running the App

To start the development server, run:

```bash
npm run start
